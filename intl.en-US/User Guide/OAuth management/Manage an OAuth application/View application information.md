# View application information {#task_266313 .task}

This topic describes how to view information about an application, such as the application name, the display name, and the application type.

1.  Log on to the [RAM console](https://ram.console.aliyun.com/).
2.  In the left-side navigation pane, click **OAuth Applications**.
3.  In the **Application Name** column, click the name of the target application.
4.  In the **Basic Information** area, view the application information, such as the application name, the display name, and the application type.
