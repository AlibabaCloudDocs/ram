# Node.js SDK 简介 {#reference_mnq_2xv_xdb .reference}

本文简要介绍了 Node.js SDK 的安装以及版本发布通知。

## Node.js SDK 的安装 {#section_hnm_fxv_xdb .section}

您可以通过 npm 安装 Node.js SDK：

``` {#codeblock_0zu_1wz_x99}
npm install aliyun-sdk
```
