# 接入地址

本文介绍了STS服务的所有接入地址，每个地址的功能都相同，请尽量在同地域进行调用。

以下表格分别罗列了各个地域的接入地址，其中接入地址一列包含如下信息：

-   公网：互联网访问地址。
-   VPC：可在同地域内VPC中访问，无需开放公网访问权限。
