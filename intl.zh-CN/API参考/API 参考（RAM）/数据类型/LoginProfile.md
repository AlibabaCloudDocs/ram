## 子节点 {#section_ldp_zh4_xdb .section}

**UserName**

-   类型：String
-   描述：用户名

**PasswordResetRequired**

-   类型：Boolean
-   描述：要求下次登录时重设密码

**MFABindRequired**

-   类型：Boolean
-   描述：要求必须绑定多因素认证设备
