## 子节点 {#section_ldp_zh4_xdb .section}

**MinimumPasswordLength**

-   类型：Integer
-   描述：最小密码长度

**RequireLowercaseCharacters**

-   类型：Boolean
-   描述：必须包含小写字母

**RequireUppercaseCharacters**

-   类型：Boolean
-   描述：必须包含大写字母

**RequireNumbers**

-   类型：Boolean
-   描述：必须包含数字

**RequireSymbols**

-   类型：Boolean
-   描述：必须包含字符
