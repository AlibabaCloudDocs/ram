# 服务地址 {#reference_sdg_3pv_xdb .reference}

## 服务地址选择建议 {#section_eys_z4v_xdb .section}

-   每个服务地址功能完全一样，请尽量同Region调用；
-   Endpoint类型
    -   公网：互联网访问地址；
    -   VPC：可在同Region内VPC中访问，无需开放公网访问权限；
